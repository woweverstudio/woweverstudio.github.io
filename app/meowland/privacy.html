MeowLand(뮤랜드) 앱

MeowLand는 로그인 없이 플레이 가능하며, 개인정보는 최소한으로만 수집하고 안전하게 관리합니다.

1. 수집하는 정보

MeowLand 앱은 다음과 같은 정보를 수집합니다:

  • 위치 정보: 날씨 정보 제공을 위해 사용자의 위치 데이터를 수집합니다.
  • 게임 데이터: 친밀도, 레벨, 아이템, 설정 등의 게임 진행 데이터는 기기 내부에 저장됩니다.

위치 정보는 사용자가 동의한 경우에만 수집되며, 날씨 정보 제공 목적으로만 사용됩니다. 위치 데이터는 서버에 저장되지 않으며 실시간으로만 사용됩니다.

2. 데이터 저장 및 보안

게임 데이터는 기기 내부 저장소(SwiftData)에 안전하게 저장됩니다. 선택적으로 iCloud 동기화를 사용할 수 있으며, 이 경우 Apple의 보안 인프라를 통해 암호화되어 저장됩니다.

위치 정보는 실시간으로만 사용되며, 사용자의 기기나 서버에 저장되지 않습니다. 날씨 API 요청 시에만 위치 데이터가 전송되며, 요청이 완료되면 즉시 폐기됩니다.

3. 제3자 서비스

MeowLand는 다음과 같은 제3자 서비스를 사용합니다:

  • 날씨 API: 위치 정보를 기반으로 날씨 데이터를 제공받습니다. 위치 데이터는 API 요청 시에만 전송되며 저장되지 않습니다.
  • xAI Grok API: 고양이 대화 번역 기능에 사용됩니다. 대화 내용은 번역 생성을 위해 전송되며 저장되지 않습니다.

이 서비스들은 각각의 개인정보 처리방침에 따라 운영됩니다.

4. 데이터 공유

MeowLand는 사용자의 개인정보를 제3자에게 판매하거나 공유하지 않습니다. 위치 정보는 날씨 API 요청을 위해서만 사용되며, 다른 목적으로 사용되거나 저장되지 않습니다. 법적 의무가 있는 경우를 제외하고 어떠한 정보도 외부로 제공되지 않습니다.

5. 사용자 권리

사용자는 언제든지 다음의 권리를 행사할 수 있습니다:

  • 위치 권한 관리: 기기 설정에서 위치 권한을 언제든지 허용 또는 거부할 수 있습니다.
  • 동의 철회: 위치 정보 수집 동의를 언제든지 철회할 수 있습니다.
  • 데이터 삭제: 앱 삭제 시 모든 게임 데이터가 함께 삭제됩니다.

위치 권한을 철회하더라도 날씨 정보를 제외한 모든 게임 기능은 정상적으로 이용할 수 있습니다.

6. 앱 권한

MeowLand는 다음과 같은 권한을 요청합니다:

  • 위치 정보 접근: 날씨 정보 제공을 위해 필요합니다. 위치 데이터는 실시간으로만 사용되며 저장되지 않습니다.

위치 권한은 선택사항이며, 사용자가 동의하지 않아도 앱의 기본 기능 사용이 가능합니다. 위치 권한을 거부하더라도 날씨 정보를 제외한 모든 게임 기능을 정상적으로 이용할 수 있습니다.

7. 광고 및 추적

MeowLand는 광고를 포함하지 않으며, 사용자 추적을 위한 어떠한 도구도 사용하지 않습니다. 사용자의 게임 플레이 패턴이나 행동을 분석하거나 추적하지 않습니다.
