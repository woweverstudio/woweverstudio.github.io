개인정보 처리방침

자유를 찾아서 앱

자유를 찾아서는 사용자의 개인정보를 수집하거나 저장하지 않습니다. 본 애플리케이션은 완전히 로컬에서 작동하며, 외부 서버로 데이터를 전송하지 않습니다.

1. 개요

Wowever Studio("회사", "저희")가 제작한 자유를 찾아서 모바일 게임 애플리케이션에 오신 것을 환영합니다. 본 개인정보 처리방침은 애플리케이션 사용 시 수집되는 정보에 대해 설명합니다.

중요: 자유를 찾아서는 사용자의 개인정보를 수집하거나 저장하지 않습니다. 본 애플리케이션은 완전히 로컬에서 작동하며, 외부 서버로 데이터를 전송하지 않습니다.

2. 수집하는 정보

자유를 찾아서는 어떠한 개인정보도 수집하지 않습니다.

  • 이름, 이메일 주소, 전화번호 등의 개인 식별 정보를 수집하지 않습니다
  • 위치 정보를 수집하지 않습니다
  • 연락처 정보에 접근하지 않습니다
  • 사진, 카메라, 마이크에 접근하지 않습니다
  • 기기 식별자를 수집하지 않습니다
  • 사용 통계나 분석 데이터를 수집하지 않습니다
  • 광고 ID를 수집하지 않습니다

3. 데이터 저장

게임 진행 상황 및 설정은 사용자의 기기에만 로컬로 저장됩니다. 이 데이터는 다음과 같습니다:

  • 게임 설정 (소리, 진동, 언어 등)
  • 게임 진행 기록 및 레벨 완료 상태 (로컬 저장만)
  • 사용자 선택 및 게임 내 결정

이러한 데이터는 오직 사용자의 기기에만 저장되며, 외부로 전송되지 않습니다. 앱을 삭제하면 모든 데이터가 함께 삭제됩니다.

4. 제3자 서비스

자유를 찾아서는 제3자 서비스를 사용하지 않습니다:

  • 광고 네트워크 없음
  • 분석 서비스 없음
  • 소셜 미디어 통합 없음
  • 클라우드 저장소 없음
  • 추적 도구 없음

저희는 사용자의 프라이버시를 최우선으로 생각합니다. 어떠한 제3자와도 사용자 정보를 공유하지 않습니다.

5. 어린이 프라이버시

자유를 찾아서는 모든 연령대에 적합한 게임입니다. 어떠한 개인정보도 수집하지 않으므로, 13세 미만 어린이를 포함한 모든 사용자의 개인정보에 대한 우려가 없습니다.

6. 권한

자유를 찾아서는 다음과 같은 최소한의 권한만을 요청합니다:

  • 저장소 접근: 게임 진행 상황을 로컬에 저장하기 위해 필요
  • 진동: 게임 피드백을 위해 사용 (설정에서 비활성화 가능)

이러한 권한은 게임 기능을 위해서만 사용되며, 어떠한 데이터도 수집하지 않습니다.

7. 데이터 보안

저희는 어떠한 개인정보도 수집하지 않으므로, 데이터 유출의 위험이 없습니다. 모든 게임 데이터는 사용자의 기기에만 안전하게 저장됩니다.

8. 사용자 권리

개인정보를 수집하지 않으므로:

  • 삭제 요청이 필요하지 않습니다 (모든 데이터는 기기에만 저장됨)
  • 데이터 이전 요청이 필요하지 않습니다
  • 언제든지 앱을 삭제하여 모든 로컬 데이터를 제거할 수 있습니다

9. 국제 데이터 전송

자유를 찾아서는 어떠한 데이터도 외부 서버나 다른 국가로 전송하지 않습니다. 모든 정보는 사용자의 기기에만 보관됩니다.

10. 개인정보 처리방침 변경

본 개인정보 처리방침은 필요에 따라 업데이트될 수 있습니다. 중요한 변경사항이 있을 경우, 앱 내 알림을 통해 공지하고 "최종 업데이트" 날짜를 갱신하겠습니다.
